I have been working on Semoss as a designer, developer, and data visualizer. Semoss is an context aware analytics tool that allows you to identify transitive relationships. Much of my work has been building a web application using angular and creating data visualizations using D3. I have also done a lot of design work including icon, brand, and interface design.
